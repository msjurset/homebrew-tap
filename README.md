# Homebrew Tap

Homebrew formulae for [runbook](https://github.com/msjurset/runbook).

## Install

```sh
brew tap msjurset/tap
```

### CLI only

```sh
brew install runbook
```

### Mac app (includes CLI)

```sh
brew install --cask runbook-mac
```

## Upgrade

```sh
brew upgrade runbook
brew upgrade --cask runbook-mac
```
